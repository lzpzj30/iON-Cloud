## iON Cloud 介绍

说到iON Cloud，这是老牌商家Krypt在2019年推出的VPS子品牌。Krypt是什么来头呢？这是1998年成立的老字号，做了二十多年服务器托管生意，业内常说的"KT机房"指的就是它。

iON Cloud专门针对亚太地区特别是中国大陆市场做了优化。机房分布在美国洛杉矶、圣何塞、达拉斯、夏威夷檀香山，还有新加坡。网络线路方面，美国机房主要走CN2 GT和联通AS4837优化线路，部分节点还有CN2 GIA；新加坡机房则是CN2直连，整体速度相当不错。

产品特色挺明显的：KVM虚拟化，SSD硬盘，1Gbps带宽起步，而且流量计算方式比较良心——只算出站流量，这在行业里算是少见的实惠了。支持Linux和Windows系统（洛杉矶、圣何塞、夏威夷机房提供Windows），付款方式支持支付宝、PayPal、信用卡，还有中文界面，国内用户用起来没什么门槛。

控制面板功能齐全，重建、重启、VNC控制台、ISO挂载、救援模式、密码修改、自助续费这些基本功能都有。这个背景和配置，用来建站或者做点其他项目，稳定性是有保障的。

## iON Cloud 测评

### 性能表现

从多个用户的实际测评来看，iON Cloud的硬件性能中规中矩。CPU采用Intel E5-2690 v3系列，主频2.6GHz，在UnixBench测试中得分处于同价位VPS的平均水平。SSD硬盘IO读写速度在200-250 MB/s左右，日常使用完全够用。

内存和CPU资源比较充裕，不会出现超售严重导致性能打折的情况。这一点跟Krypt的老牌背景有关系，机器管控相对严格。

### 网络速度

网络这块是iON Cloud的强项。洛杉矶和圣何塞机房三网回程都走联通AS4837优化线路，电信用户白天晚上都比较稳定，南方电信用户反馈速度甚至能跟搬瓦工DC6机房媲美。

具体测试数据来看：
- 电信用户走CN2 GT或CN2 GIA出国，回程AS4837，延迟150-180ms左右，晚高峰下载速度能保持在500KB/s-1MB/s
- 联通用户全程AS4837骨干网直连，延迟和速度都很不错
- 移动用户走Level 3转接，相对来说会稍微慢一点，平均延迟220ms左右

新加坡机房采用CN2直连，到国内延迟更低，大概在50-80ms，适合对延迟敏感的应用。

有个特别实在的地方：流量只计算出站方向。市面上大部分VPS都是双向计费，iON Cloud这个单向流量政策算是业界良心了。你下载多少都不算在流量里，只有从服务器往外传数据才计费。

### 稳定性

Krypt运营了20多年，机房和网络维护都比较成熟。从用户反馈看，iON Cloud的稳定性还可以，适合建站使用。偶尔会有维护导致的短暂中断（比如之前有过因为烟雾报警停电的情况），但总体来说问题不大。

24/7的客服支持也都在，遇到问题可以及时解决。售后响应速度在同类商家中算是靠前的。
